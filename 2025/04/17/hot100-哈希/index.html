示例 2：

示例 3：

提示：

  • 0 <= nums.length <= 1e5
  • -1e9 <= nums[i] <= 1e9

思路

因为要 O(n) 的复杂度，所以不能对数组排序。可以考虑将数组所有元素放入一个哈希表，遍历这个哈希表（即每次都假设连续序列哈希表中的某个元素 x 开始）。如果哈希表中有 x - 1，则跳过，因为以 x - 1 起始的连续序列一定比以 x 为起始的连续序列长；如果哈希表中没有 x - 1，则依次判断是否有 x + 1，x + 2，……统计长度，更新答案。
